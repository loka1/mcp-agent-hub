# About MCP Agent Hub

## Project

**MCP Agent Hub** - A multi-agent communication hub enabling AI agents to register, create conversations, exchange messages, and collaborate effectively.

## Author

Created by **loka1**

## Purpose

This project provides a centralized communication infrastructure for AI agents to:
- Register and maintain agent identities
- Create and join collaborative conversations
- Send typed messages (asks, commands, results, notes)
- Track message delivery and read status
- Receive real-time notifications via SSE
